# AI Electronics Lab — Evidence-First Circuit Simulation Agent

A compact, reproducible Kaggle capstone edition of AI Electronics Lab.

The system converts a plain-language electronics request into:

1. a structured circuit plan;
2. deterministic validation;
3. a deterministic SPICE netlist;
4. an ngspice simulation;
5. analytical verification;
6. plots and downloadable evidence;
7. an explanation constrained to verified results.

## Initial supported scope

- RC low-pass filter
- RC high-pass filter
- resistive voltage divider

This repository is currently in its specification and extraction phase.

No production credentials, services, databases, or infrastructure are required.
