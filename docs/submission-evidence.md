# Submission evidence

Record only reproducible and non-sensitive evidence here.

## Required evidence

- clean-clone installation;
- local application startup;
- supported prompt execution;
- ngspice output;
- generated schematic and plot;
- PASS, WARN, and FAIL examples;
- what-if comparison;
- automated test results;
- evaluation results;
- secret-scan results;
- meaningful Codex contribution;
- meaningful Antigravity contribution;
- final video timestamps and screenshots.

Do not include hidden chain-of-thought, credentials, or private production traces.
